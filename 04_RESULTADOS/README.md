# Results note

Los resultados oficiales de la replica HHI/DEA se generan en `outputs/tables/`,
`outputs/figures/` y `paper/`.

La carpeta `04_RESULTADOS/` queda reservada para compilaciones auxiliares del
proyecto vigente y para trazabilidad interna. No debe contener resultados de
lineas tematicas ajenas al nucleo de esta replica.

Si un resultado cuantitativo no aparece en `outputs/` o `paper/`, debe tratarse
como auxiliar o `PENDIENTE_VERIFICAR`.
