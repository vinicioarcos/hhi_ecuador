# Protocolo de no fabricacion

## Regla

El proyecto replica el paper original y extiende a Ecuador. Ningun resultado nuevo puede escribirse en manuscritos, slides, dashboard o submission si no existe en una salida reproducible del pipeline o en una fuente oficial verificable.

## Aplicacion

1. Para HHI, citar `outputs/tables/hhi_replication_comparison.csv` o las tablas publicables derivadas.
2. Para DEA, citar `outputs/tables/dea_models_summary.csv` y tablas de auditoria/publicacion.
3. Para Ecuador, citar `outputs/tables/hhi_ecuador.csv` y resultados DEA asociados.
4. Para revista objetivo, citar la web oficial del MLAJ y dejar la indexacion Latindex como `PENDIENTE_VERIFICAR` si no hay evidencia oficial.
